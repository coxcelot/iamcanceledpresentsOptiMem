🚀 Setup Instructions

Part A: Colab Translation Server

1Upload the Colab Notebook

Go to Google Colab and upload optmem_translation_server.ipynb from the ZIP.

2Run All Cells

Run the three cells in order:

# Cell 1: Install Argos Translate
!pip install argostranslate
!argostranslate package install translate-en_zh

# Cell 2: Register translation bridge
import argostranslate.translate as at
from google.colab import output

translator = at.get_translation_from_codes("en", "zh")

def translate_text(text, protect_terms=None):
    # Protects technical terms, translates, restores
    return translator.translate(text)

output.register_callback("optmem.translate", translate_text)

# Cell 3: Keep-alive
import time
while True:
    time.sleep(60)

⚠ Keep this Colab tab open while using the extension. The runtime must stay active.

Part B: Browser Extension

3Load the Extension

1. Extract the ZIP to a folder on your computer

2. Open Chrome and go to chrome://extensions

3. Enable Developer mode (top-right toggle)

4. Click Load unpacked and select the extension/ folder

5. The OptiMem icon appears in your toolbar

4Configure (Optional)

Right-click the extension icon → Options to set:

  • Auto-compress threshold (default: 30,000 tokens)
  • Canvas size (default: 1568px — Claude's optimal resolution)
  • Font size (default: 10px — safe for OCR; 8px = denser but riskier)

Part C: Using OptiMem

5Compress a Conversation

1. Open a conversation in Claude, ChatGPT, or Gemini

2. Click the OptiMem extension icon

3. Verify Colab status shows ✓ Connected

4. Click Analyze to see estimated token savings

5. Click Fetch & Compact to run the full pipeline

6. Images + memory_pack.json download to ~/Downloads/optmem/

6Use the Memory Pack

Send the downloaded images to any vision-capable LLM:

# Claude API
messages=[{
  "role": "user",
  "content": [
    {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": base64_data}},
    {"type": "text", "text": "Here is our conversation history as images."}
  ]
}]

# OpenAI API
messages=[{
  "role": "user",
  "content": [
    {"type": "image_url", "image_url": {"url": "data:image/png;base64," + base64_data}},
    {"type": "text", "text": "Here is our conversation history as images."}
  ]
}]

⚙ Configuration Reference

Canvas Size

1568x1568

Claude bills ~3278 tokens per image. Optimal for vision OCR.

Font Size

10px monospace

33,930 chars per image. 8px = 53K chars but OCR risk.

Crossover Threshold

~13,112 chars

Minimum prose chars for optical to save tokens.

Auto-Compress

30K tokens

Triggers every 30s when conversation exceeds threshold.
